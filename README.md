# knu_washer

학우들에게 실제로 도움이 될 앱에 대해 고민\n
구글 폼을 통한 설문조사에서 많은 학우들이 세탁기 앱에 대한 필요성을 느끼고 있다고 답변\n
학우들이 스마트폰으로 세탁기의 상태를 확인할 수 있는 앱을 제작
